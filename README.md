# homebrew-port-port

Homebrew tap for [port-port](https://github.com/ljack/port-port) CLI tools.

## Install

```sh
brew tap ljack/port-port
brew install port-port
```

## What's included

- **`port-port-cli`** — Interactive CLI + TUI for port monitoring
- **`port-port-mcp`** — MCP server for Claude Code integration
